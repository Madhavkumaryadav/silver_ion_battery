# Failure Modes and Update Layout

## Failure Modes and Corrective Actions

- Slurry sedimentation too fast:
  Action: increase homogenization quality, reassess solids loading, and revisit binder proportion inside allowed range.

- Coating cracks after drying:
  Action: reduce stress in drying profile, adjust binder fraction, and verify solvent removal rate.

- High first-cycle irreversible loss:
  Action: review electrolyte wetting completeness, rest durations, and formation current profile.

- Unstable pulse-response fitting:
  Action: improve data sampling quality, increase pulse-rest protocol consistency, and remove outlier runs.

## Suggested Doc Section Layout for Future Updates

Use this section layout whenever you add a new chemistry iteration:

1. Objective of iteration
2. Material set and lot numbers
3. Ratio and slurry settings
4. Coating and drying settings
5. Assembly and electrolyte notes
6. Formation protocol
7. Test summary
8. Extracted parameters
9. Issues and corrective actions
10. Decision for next iteration
